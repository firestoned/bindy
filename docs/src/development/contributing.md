# Contributing

Thank you for contributing to Bindy!

## Ways to Contribute

- Report bugs
- Suggest features
- Improve documentation
- Submit code changes
- Review pull requests

## Getting Started

1. [Set up development environment](./setup.md)
2. Read [Code Style](./code-style.md)
3. Check [Testing Guidelines](./testing-guidelines.md)
4. Follow [PR Process](./pr-process.md)

## Code of Conduct

Be respectful, inclusive, and professional.

## Reporting Issues

Use GitHub issues with:
- Clear description
- Steps to reproduce
- Expected vs actual behavior
- Environment details

## Feature Requests

Open an issue describing:
- Use case
- Proposed solution
- Alternatives considered

## Questions

Ask questions in:
- GitHub Discussions
- Issues (tagged as question)
